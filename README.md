# Business Sales Performance Analytics
# Overview
This project analyzes Superstore sales data to identify revenue trends, top-selling products, profitable categories, and regional performance.

* Python
* Google Colab
* Pandas
* Matplotlib
* Seaborn
* GitHub
# Key Analysis
* Revenue Trend Analysis
* Top Selling Products
* Category-wise Performance
* Regional Performance
* Discount vs Profit Analysis
# Key Insights
* Identified top-performing categories.
* Analyzed regional sales performance.
* Evaluated discount impact on profits.
* Generated business recommendations.
